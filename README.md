# IUS EX MACHINA

Juego juridico didactico para preparar el Examen de Grado de Derecho chileno.

## Deploy en Vercel

Este proyecto es una app estatica: `index.html` contiene el juego completo.

En Vercel:

1. Importa el repositorio desde GitHub.
2. Framework preset: `Other`.
3. Build command: vacio.
4. Output directory: `.`.
